FAQ

What is a linked task?

A linked task is one which is linked to an existing asset/insurance/loan. Linked tasks allow you to work seamlessly without having to worry about entering the details of the linked asset/insurance/loan manually. Secondly, you can track the associated advice/action plan immediately.

What is a non-linked task?

A non-linked task, as the name suggests, is the opposite of a linked task. There are no assets/insurance/loans associated with such tasks. Depending the nature of your work environment, you can create such task templates. E.g. of non-linked tasks: On-board a new cient, fill up form, do a review meeting etc.

A note on tasks

It is worth noting that tasks allows you to cordinate between you and your team only. The task interface is not opened up to your clients.

Tasks can also have sub-tasks associated with them. You can assign default owners and turn around times to tasks as well as sub-tasks.
