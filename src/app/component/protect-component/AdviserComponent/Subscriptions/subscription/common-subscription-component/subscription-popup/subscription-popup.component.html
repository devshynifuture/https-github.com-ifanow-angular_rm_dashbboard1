What is Subscriptions?

Subscriptions is carefully designed for RIAs. You can define your core offerings,
fee structure, deliverables, agreements and automate the whole process around it.
This will act as a foundation for your RIA practice.

Create Plans

You start by creating Plans first.Plans are you core offering. You can have
multiple plans for different type of clients Each plan will have sub offerings,alse
known as Services(Explain in next slide)

Create Services

Next up, you create Services. More than one Service can be added to a Plan. E.g.
Plan could be Financial Planning. It’s Services could be 1) Planning fees 2) Implementation services 3) Retainer services.

Create Documents

We’ve automated the complete process of documentation (a core part of RIA
compliance). You create documents easily using ready made templates or upload. Email to clients, get eSigned and securely store.

Map Plans, Services & Documents

Once you’ve created the above, you simply map Plans, Services & Documents to
each other. This will result in a cohesive product offering which is unique to your practice and ensures stability and consistency.

Plan, rinse, repeat!

With the foundation now set, it’s time to now do what you do the best - Plan for
your clients’ financial freedom. We do the background work for you - create invoices, bill clients, execute agreements and so on.

Create document

Start by creating documents easily using ready made templates, create your own or upload

Send to client

Once you create the document, email it to client with one click

Client inspects document

Client can review the document and proceed for eSigning

Client performs Aadhaar eSign

Using Aadhaar based eSign client signs the document which is legally valid in court of law

Securely store

All documents are securely stored, giving you audit trail for future compliance
