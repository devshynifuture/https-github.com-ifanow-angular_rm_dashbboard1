HOW E-SIGN PROCESS WORKS

1 The user opens the eSigning link from his/her email or text message. The eSigning link can also be opened from within the application.

2 The user views the document and clicks on ‘proceed’.

3 The user gives consent for electronic execution, and clicks on ‘Sign’

4 User enters Aadhaar Number/VID; clicks on the check-box and requests for OTP.

5 The user enters the OTP and clicks on ‘Submit’.

6 The document has been signed. The user is redirected back to your application.

7 You’ll be notified on the subscription dashboard once the document has been signed.
